This read me is to instruct how to compile, execute and run the 2 contained programs. One which is
done in java and one in objective c. 
They were developed on windows 7 in using Ant and GnuStep.

FOR JAVA
1. open bash and mount the directory Assign0
2. run ant execute

FOR OBJECTIVE C
1. open bash and mount the directory Assign0
2. run ./gnustepcompile
3. run ./MyRectangle.exe
